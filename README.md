# APP PAES 2025

Aplicación web para la preparación de la PAES Regular 2025.

## Características
- **Ensayos Interactivos**: Simulación real con cronómetro y puntaje.
- **Material Didáctico**: Contenido organizado por ejes temáticos.
- **Progreso**: Seguimiento detallado del rendimiento.
- **Diseño Responsivo**: Adaptable a móviles y tablets.

## Tecnologías
- Next.js 14
- TypeScript
- CSS Modules
- Vercel Postgres (Producción) / JSON Local (Desarrollo)
- NextAuth.js (Autenticación)

## Instalación
1. Clonar el repositorio.
2. Instalar dependencias: `npm install`.
3. Ejecutar servidor de desarrollo: `npm run dev`.
